Nota Fiscal de Políticos - DF

Este site foi criado para oferecer transparência sobre os gastos dos deputados do Distrito Federal. Aqui você pode:

  • Comparar os gastos de dois deputados diferentes
  • Filtrar os gastos por tipo de despesa
  • Visualizar dados atualizados diretamente da API da Câmara

Use o site para se informar melhor e acompanhar como os recursos públicos estão sendo utilizados.
