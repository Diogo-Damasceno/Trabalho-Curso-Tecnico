Tecnologias Utilizadas

Este site foi desenvolvido utilizando as seguintes tecnologias:

  • HTML + CSS: para estrutura e estilo das páginas.
  • JavaScript: para manipulação de dados e interatividade com o usuário.
  • API de Dados Abertos da Câmara dos Deputados: para obter informações atualizadas dos deputados federais do DF.
  • GitHub Pages: para hospedagem gratuita e fácil do site.

Todo o conteúdo é atualizado dinamicamente usando requisições à API pública da Câmara.
